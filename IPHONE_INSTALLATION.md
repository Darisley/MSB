# 📱 Guia de Instalação no iPhone

## ✅ Pré-requisitos
- iPhone com iOS 12.2 ou superior
- Safari instalado (pré-instalado no iOS)
- Conexão com internet (para primeira instalação)

---

## 🚀 Passos para Instalar como App

### **1️⃣ Abrir no Safari**
```
Acesse: https://darisley.github.io/MSB
```

### **2️⃣ Menu Compartilhar**
- Toque no ícone **⬆️ (Compartilhar)** - está na barra inferior
- Se não encontrar, toque em **menu ⋯** (canto superior direito)

### **3️⃣ Adicionar à Tela de Início**
- Veja a opção **"Adicionar à Tela de Início"**
- Toque nela

### **4️⃣ Nomeie o App** (Opcional)
- Você pode mudar o nome padrão
- Exemplo: "Controle EPI" ou "MSB"
- Toque em **"Adicionar"**

### **5️⃣ Pronto!** ✅
- O app aparecerá na sua **home screen**
- Você pode movê-lo para qualquer pasta
- Toque para abrir como app nativo

---

## 🎯 Funcionalidades no iPhone

✅ **Funciona Offline** - Sem internet não há problema  
✅ **Tela Cheia** - Sem barra de navegação do Safari  
✅ **Ícone na Home** - Acesso rápido como qualquer app  
✅ **Notificações** - Notificações visuais integradas  
✅ **Dados Locais** - Tudo salvo no seu iPhone  
✅ **Dark Mode** - Alterna entre tema claro/escuro  

---

## 🔄 Atualizações

Quando você fizer push de atualizações no GitHub, o app baixará automaticamente:
1. A versão em cache será usada (offline)
2. Caso haja atualizações, o service worker as carregará
3. Na próxima visita, terá a versão mais recente

---

## 🤔 Dúvidas Frequentes

### **P: O app funciona sem internet?**
✅ Sim! Tudo funciona offline. Seus dados são salvos no iPhone.

### **P: Posso compartilhar dados entre dispositivos?**
✅ Sim! Faça backup como JSON e compartilhe via Email/AirDrop.

### **P: O app fica visível no multitarefa?**
✅ Sim! Funciona como qualquer app nativo do iOS.

### **P: Posso deletar e reinstalar?**
✅ Sim! Toque seguro no ícone > Remover App. Depois reinstale.

### **P: Meus dados somem se deletar o app?**
❌ Seus dados locais ficarão salvos enquanto você não limpar o Safari cache.

---

## 📞 Suporte

Se encontrar problemas:
1. **Limpe o cache do Safari** - Configurações > Safari > Limpar Histórico e Dados
2. **Reinstale o app** - Delete e adicione novamente
3. **Atualize o iOS** - Garanta que está na versão mais recente
4. **Teste no modo normal** - Não funciona em abas privadas
