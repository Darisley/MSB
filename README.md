# MSB Construções - Controle de EPI (PWA)

Sistema completo para controle de entrega de Equipamentos de Proteção Individual (EPI), agora como Progressive Web App (PWA) pronto para instalação em qualquer dispositivo.

## 📱 Funcionalidades

- ✅ Cadastro de funcionários com importação Excel
- ✅ Cadastro de EPIs
- ✅ Importação de notas fiscais (XML, CSV, TXT)
- ✅ Registro de entregas com assinatura digital
- ✅ Múltiplos EPIs por entrega
- ✅ Relatórios em PDF e Excel
- ✅ Dashboard com estoque baixo
- ✅ Funciona offline
- ✅ Instalável como app no celular
